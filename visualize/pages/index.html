    Twitter Streaming

    Implemented Twitter Streaming using Apache Spark to collect tweets with hashtag #NowPlaying

  • Data Cleansing

    Preprocessing: Remove noise from tweets

    Preprocessed data to identify songs and artists being talked about in the tweets. Made some Groovy tweets.

  • More Data!

    iTunes Search API integration

    Identify genres for artists and/or songs identified in the preprocessed tweets using iTunes Search API and Apple iTunes Enterprise Partner Feed (disk based caching of request responses to minimize network IO)

  • Data Analysis

    Tweets be tellin' us like..

    The most popular genres in any given hour, the 10 most popular songs and artists in genres being talked about, etc.
